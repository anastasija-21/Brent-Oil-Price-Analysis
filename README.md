# Brent-Oil-Price-Analysis
This project analyses historical Brent crude oil prices using data from the U.S. Energy Information Administration, covering the period from 1987 to 2026. It explores price trends through time-series analysis, including moving averages (7-day and 30-day), return calculations, and volatility measures.
